# ai-sandbox

To re-create the python env use:
python3 -m venv .venv_name (minimum python version 3.8)

pip install -r requirements.txt

Then the usual:
source venv_pytorch/bin/activate or deactivate


The code is simple enough that should work with newer versions of the requirements which are:
torch
numpy
pandas
huggingface_hub, transformers, PEFT
